# BookSprout — приложение-каталог

Веб-витрина (booksprout.kz) + устанавливаемое приложение на телефоне (PWA). Корзины и оплаты нет — каждая кнопка ведёт в WhatsApp с готовым текстом.

## Структура файлов

```
booksprout-app/
├── index.html              — всё приложение (товары, стили, логика)
├── img/                    — фотографии товаров (25 фото)
├── manifest.webmanifest    — делает сайт устанавливаемым PWA
├── sw.js                   — офлайн-кэш (нужен для установки)
├── icon-192.png            — иконка приложения
├── icon-512.png            — иконка приложения (большая)
└── README.md
```

## Каталог

30 товаров. Фильтры: **все**, **0–2**, **2–4**, **4–6**, **наклейки**, **flash cards**.

| Тип | Кол-во | С фото |
|-----|--------|--------|
| Книги (board books, клапаны, музыкальные, магнитные, тактильные) | 21 | 21 |
| Flash Cards (карточки) | 6 | — |
| Наклейки (sticker books) | 2 | 1 |
| Игрушки (Hape, дерево) | 1 | 1 |

Сортировка на сайте:
1. В наличии + с фото — сверху
2. В наличии + без фото — ниже
3. Под заказ — внизу, затемнённые (opacity 0.45 + бейдж «под заказ»)

## Что отредактировать

Открой `index.html`, найди блок **«РЕДАКТИРУЙ ТОЛЬКО ЭТОТ БЛОК»** в `<script>`.

### Добавить товар

Скопируй любую строку в `PRODUCTS` и поменяй поля:

```js
{id:"goodnight", name:"Goodnight Moon", sub:"перед сном", price:7600, cats:["2-4"], img:"img/goodnight.jpg", tier:"", stock:true},
```

| Поле | Описание |
|------|----------|
| `id` | Уникальный (латиница, без пробелов) |
| `name` | Название, видит клиент |
| `sub` | Короткая подпись под названием |
| `price` | Число в тенге (`7600`, не `7 600`) |
| `cats` | Теги для фильтров — можно несколько через запятую |
| `img` | Путь к фото (`"img/file.png"`). Пусто `""` = иконка-заглушка |
| `tier` | `"target"` = метка «выбор». Иначе `""` |
| `stock` | `true` = в наличии. `false` = под заказ (карточка затемнена) |

Доступные теги для `cats`:

| Тег | Фильтр |
|-----|--------|
| `"0-2"` | 0–2 года |
| `"2-4"` | 2–4 года |
| `"4-6"` | 4–6 лет |
| `"stickers"` | наклейки |
| `"flash-cards"` | flash cards |

Товар может иметь несколько тегов: `cats:["0-2","2-4","flash-cards"]`.

### Добавить фото

1. Положи файл (png/jpg/jpeg) в папку `img/`
2. В `PRODUCTS` укажи путь: `img:"img/FileName.png"`

### Поменять номер WhatsApp

```js
const WHATSAPP = "77005530007";
```

Формат: `77011234567` (без `+` и пробелов).

### Добавить категорию

Добавь строку в `CATEGORIES`:

```js
{key:"6-8", label:"6–8"},
```

И используй `"6-8"` в `cats` нужных товаров.

## Как опубликовать (бесплатно)

**Netlify Drop** — [netlify.com/drop](https://netlify.com/drop) → перетащи всю папку → готовая ссылка за минуту. Потом подключи домен booksprout.kz.

**GitHub Pages** — залей папку в репозиторий → Settings → Pages → деплой из main.

PWA (установка, офлайн) работает только по **https**. На Netlify/GitHub Pages https включён сразу.

## Ссылка на каталог в Instagram

1. Опубликуй на хостинг — получишь ссылку
2. Instagram → профиль → «Редактировать» → поле «Веб-сайт» → вставь ссылку
3. В Stories — стикер «Ссылка», закрепи Highlight «Каталог»

## Установка приложения

- **Android / Chrome:** баннер «Добавить на главный экран» или меню ⋮ → «Установить приложение»
- **iPhone / Safari:** кнопка «Поделиться» → «На экран Домой»

Иконка booksprout на телефоне открывает витрину без адресной строки — выглядит как приложение.
